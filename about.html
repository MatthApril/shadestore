About Us

Our Story

Welcome to Shade store! My name is Matthew Aprilian, from ISTTS. This store was built with immense dedication and effort to create a smooth and enjoyable experience for users. From conceptualizing the design to implementing functionalities, every step of the journey required meticulous planning.

This website represents a culmination of learning and practical application of modern web technologies, including HTML, CSS, JavaScript, and Bootstrap. By incorporating user-friendly navigation, aesthetic design, and seamless functionality, we aim to provide our visitors with a top-notch experience.

How It Was Made

Creating this store involved hours of coding, debugging, and refining details. The website employs:

  • Responsive design to ensure accessibility across all devices.
  • Interactive features built with JavaScript to enhance user engagement.
  • The Bootstrap framework to maintain a consistent and professional appearance.

We took inspiration from real-world e-commerce platforms, analyzing their strengths and weaknesses to build a unique project that blends creativity with practicality.

Penjelasan Efek Interaktif "Thank You"

Efek interaktif "Thank You" ini dibuat menggunakan JavaScript untuk menciptakan pengalaman visual yang dinamis dan menarik di halaman About Us. Berikut adalah penjelasan code JavaScript saya:


1. DOMContentLoaded
Menunggu hingga seluruh dokumen HTML selesai dimuat sebelum menjalankan kode di dalam fungsi ini.
2. const messages
Mendefinisikan array messages yang berisi kumpulan pesan "terima kasih" dalam berbagai bahasa. Array ini akan digunakan untuk menghasilkan elemen teks pada word cloud.
3. const cloudContainer
Mengambil elemen HTML dengan ID word-cloud sebagai kontainer utama untuk word cloud.
4. const containerWidth dan containerHeight
Mengambil lebar dan tinggi dari word cloud container
5. messages.forEach((message) => {
const word
...
});
Mengiterasi setiap elemen dalam array messages. Const word akan membuat elemen div baru untuk menampilkan kata, Memberikan kelas CSS thank-you pada elemen tersebut untuk styling, dan mengisi elemen div dengan teks dari elemen array messages.
6. const x dan y
Menghitung posisi acak untuk setiap kata berdasarkan ukuran kontainer. Math.random() akan menghasilkan angka acak antara 0 dan 1. x adalah posisi horizontal (kiri) acak dengan batas pada lebar kontainer dikurangi 100 piksel. y adalah posisi vertikal (atas) acak dengan batas pada tinggi kontainer dikurangi 50 piksel. Posisi acak ini diterapkan ke properti CSS left dan top.
7. cloudContainer.appendChild(word);
Menambahkan elemen div yang berisi kata ke dalam kontainer word cloud (cloudContainer).
8. function AnimateWords
Mendefinisikan fungsi untuk menggerakkan kata-kata di dalam word cloud. words akan mengambil semua elemen dengan kelas thank-you. Untuk setiap elemen, newX dan newY akan menghitung posisi acak baru. transform akan menggeser elemen ke posisi baru menggunakan properti CSS translate. transition akan menambahkan efek transisi selama 3 detik menggunakan easing ease-in-out.
9. setInterval(animateWords, 3000);
Memanggil fungsi animateWords setiap 3 detik untuk menggerakkan kata-kata ke posisi acak baru secara periodik.

Sekian dan Terima Kasih.
